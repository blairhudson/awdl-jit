AWDL-JIT

One-command setup for the AWDLControl workflow recommended by NVIDIA for GeForce NOW on Mac.

curl -fsSL https://blairhudson.com/awdl-jit/run-latest.sh | bash

What It Does

  • Creates NVIDIA GeForce NOW (AWDL).app in ~/Applications
  • Starts AWDLControl.app before launching GeForce NOW through the generated launcher
  • Stops AWDLControl.app when GeForce NOW quits, if AWDL-JIT started it
  • Adds a fallback watcher for direct GeForce NOW launches

Requirements

  • macOS
  • Installed /Applications/GeForceNOW.app
  • Installed AWDLControl.app
